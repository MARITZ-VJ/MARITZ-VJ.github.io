# Professional Portfolio | MARITZ-VJ

The official web portfolio for **Vukona Jubby Maritz**, showcasing software engineering projects, professional certifications, and technical expertise.

Live at: **[maritz-vj.github.io](https://maritz-vj.github.io/)**

---

## Technical Highlights
* **Modern Frontend:** Built with semantic HTML5, CSS3 (Custom Properties/Variables), and Vanilla JavaScript.
* **Responsive Engine:** Fully adaptive layouts with mobile-first design and a custom-built modal system.
* **Performance:** Optimized assets and video backgrounds for fast load times.
* **Interactive UI:** Features a dynamic Dark Mode toggle, typed-text effects, and project filtering by category.
* **Analytics & Contact:** Integrated Google Analytics (Gtag) and EmailJS for seamless communication.

---

## Project Structure
* `/css`: Modular stylesheets (style.css for layouts, components.css for reusable UI).
* `/js`: Core logic for dark mode, project filtering, and sliders.
* `/assets`: Optimized project documentation (CV/Certificates) and media.

---

## Deployment
Hosted via **GitHub Pages** with automated deployment from the `main` branch.
